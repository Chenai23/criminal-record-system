CRIMINAL RECORD MANAGMENT SYSTEM

The Criminal Record Management System is designed to replace the traditional method of using hardcopies, where Personnel had to manually go through folders and piles of paper to obtain a record.

The system relies on a shared database, in which data entered in the Admin Dashboard is available for regular users to access in the User Dashbaord. The system allows for an admin user to grant and revoke access, as well as add and edit records.

The system aims to digitilize and provide a quick yet secure method of obtaining Criminal Records. Making the job of Police Officers more easier and less time consuming.
